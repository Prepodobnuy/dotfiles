### hyprland pywal dotfiles
## What does pywal affect?
- qt apps
- vesktop 
- alacritty
- rofi
- swaync
- but not gtk3/gtk3+/gtk4 apps ;(
## Installation
first of all install [rpaper](https://github.com/Prepodobnuy/rpaper), and then:
```bash
git clone https://github.com/Prepodobnuy/dotfiles.git
cd dotfiles
chmod +x install.sh
./install.sh
cd ..
rm -rf dotfiles
```
